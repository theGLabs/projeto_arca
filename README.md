# 🐾 Programa ARCA
### Plataforma de Bem-Estar Animal — Prefeitura da Serra · ES

Aplicação web desenvolvida para o Programa ARCA, iniciativa oficial da Prefeitura da Serra voltada à proteção, cuidado e bem-estar animal.

---

## 📋 Funcionalidades

- 🔐 Login e cadastro com validação de e-mail
- 🐶 Listagem de animais disponíveis para adoção com filtros e paginação
- 📋 Formulário completo de adoção
- ✂️ Agendamento de castração
- 🚑 Solicitação de resgate
- ⚠️ Denúncia de maus-tratos (anônima)

---

## 🛠 Tecnologias

- HTML5, CSS3, JavaScript
- Bootstrap 5.3
- Fontes: Fraunces + DM Sans (Google Fonts)
